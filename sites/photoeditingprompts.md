- Long Description EN:
  PhotoEditingPrompts is a curated prompt library for AI photo editing. It collects and tests copy-ready prompts for ChatGPT Image — organized by task (headshots, product photography, old photo restoration, background editing, style transfers) and by scenario (LinkedIn photo, white-background product shot, corporate resume, cinematic grade).

  Every prompt is annotated: what the edit does, what details to preserve, and what tends to go wrong — so users get predictable results without trial and error. Each prompt page includes the full copyable text and side-by-side before/after examples.

  The site is built as an editorial reference, not a marketplace or SaaS tool. It is image-first: photography leads, text annotates. The design is warm and quiet — meant to feel like a well-kept field journal rather than a tools directory.

  Editorial policy: every prompt is tested before publishing. The site does not accept paid placements or sponsored listings.
