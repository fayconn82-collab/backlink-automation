主要知识库在 gbrain 维护，此仓库作为本地文件备份和版本控制。

## 目录结构

```
├── sites/              # 网站资料（每个站一个 .md）
├── assets/             # Logo + 截图
├── records/daily/      # 每日提交记录
├── platforms/          # 外链平台列表
└── skills/             # 自动化 Skill 文件
```
